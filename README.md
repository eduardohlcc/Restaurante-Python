# Sistema de Restaurante em Python

Sistema desenvolvido em Python para gerenciamento de restaurante, permitindo cadastrar pratos, gerenciar mesas e verificar listas de itens.
## Atualização 1.0 - Cardápio e Mesas!

* Criação do sistema de cardápio com cadastro, listagem e remoção de pratos!
* Criação do sistema de mesas com abertura e fechamento de mesas!

## Futuras Atualizações

* Salvar dados para não perder informações ao fechar o programa
* Criar relatório de faturamento do dia
* Criar uma interface FrontEnd para o projeto

## Funcionalidades

* Cadastro de pratos no cardápio
* Listagem de pratos com status de disponibilidade
* Remoção de pratos
* Cadastro de mesas
* Abertura e fechamento de mesas
* Menu interativo no terminal

## Tecnologias utilizadas

* Python
* VS Code
* Git e GitHub

## Propósito do projeto

Este projeto foi criado com o objetivo de praticar lógica de programação, funções, listas, dicionários e desenvolvimento de sistemas utilizando Python.
